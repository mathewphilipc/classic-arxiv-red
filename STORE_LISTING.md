# Suggested Chrome Web Store Listing

## Name

Classic arXiv Red

## Short description

Restores arXiv's classic crimson header.

## Detailed description

Classic arXiv Red brings back the familiar crimson header on arXiv.org.

The extension has one purpose: it locally changes the colors of arXiv's site
header and navigation. It contains no JavaScript, collects no data, makes no
network requests, and requests no general-purpose Chrome permissions.

The project is intentionally small and open source.

## Single purpose statement

Restore the classic crimson color of the arXiv.org header.

## Privacy declarations

- User data collection: None
- Remote code: No
- Analytics: No
- Advertising: No
